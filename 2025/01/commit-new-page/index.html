  1. 首先使用以下命令，创建一个文件，然后修改该文章： hugo new content/posts/habits.md
  2. 写好后，直接执行“hugo”
  3. 接着进入“public”目录，提交到githu: git add . git commit -m "meaasge" git push origin main
